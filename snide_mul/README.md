# SNIDE_MUL v2.5 🚀
> Advanced Interactive Bash Recon & Vulnerability Fuzzing Framework for Bug Bounty

**SNIDE_MUL** — это легковесный интерактивный комбайн, написанный на чистом Bash, предназначенный для проведения быстрой разведки (Recon/OSINT) и первичного анализа защищенности веб-ресурсов. Инструмент разработан специально для исследователей безопасности (Bug Bounty) и автоматизирует рутинные процессы сбора данных о цели.

## 🌟 Основные возможности (Core Features)

1. **Умная санитизация ввода:** Автоматически очищает входящие URL, отсекая протоколы (`http/https`) и GET-параметры, приводя цель к чистому домену.
2. **Адаптивный обход файрволов:** При обнаружении целей в зонах `.ru/.su/.рф` скрипт автоматически переключает флаги сканирования и методы верификации для обхода строгих правил фильтрации ICMP-трафика.
3. **Паспорт исследователя (Bug Bounty Header):** Все сетевые запросы (`curl`, `nmap`, `gobuster`) принудительно маркируются кастомным HTTP-заголовком `X-Bug-Bounty`. Это позволяет легитимно идентифицировать ваши сканы в логах администраторов цели.
4. **Комплексный OSINT:** Пробивка цели по базам `Whois` с автоматическим парсингом страны (Geo) и провайдера (ISP).
5. **Интегрированный фаззинг уязвимостей:**
   - **SQLi PRO:** Проверка GET-параметров на классические инъекции с URL-кодированием пейлоадов «на лету» и анализом сигнатур ошибок СУБД.
   - **XSS PRO:** Тестирование на отраженные (Reflected) XSS-уязвимости с проверкой возврата пейлоада в исходном коде страницы без фильтрации.

## 🛠️ Зависимости (Dependencies)
Для полноценной работы всех модулей скрипту необходимы:
- `nmap` — сканирование портов и определение ОС.
- `whois` — сбор регистрационных данных.
- `curl` — отправка кастомных HTTP-запросов.
- `subfinder` — поиск поддоменов (опционально).
- `gobuster` — брутфорс директорий (опционально).

## 🚀 Установка

### Вариант 1: Из AUR (Для пользователей Arch Linux)
```bash
yay -S snide-mul-git
```

### Вариант 2: Ручная установка (Универсальная)
```bash
git clone https://github.com
cd snide_mul
sudo chmod +x install.sh
sudo ./install.sh
```
